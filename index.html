CSS Topic - Colors

Colors can be supported in all browsers in the web. With CSS, colors can be specified in different ways: color name, RGB value, and Hex codes. Color will add life and light to your websites and will be used within our style sheets to add to our individual styles. All modern browsers will support 140 different colors by their color name.

Color Names

Within your CSS style sheet you will be given suggested names as you type, and even will be given the color next to the name once your color is typed in the stylesheet.

Hex Codes

Hex codes can be simplified into the value form of: #rrggbb where r is Red g is Green and b is Blue. These values are between 00 and ff. Hex codes can even be 3 digits for example: aa22ff can be written as a2f.

RGB Values

These are supported in all browsers. RGB colors values will be coded in (Red,Green,Blue) values from 0 to 255.
